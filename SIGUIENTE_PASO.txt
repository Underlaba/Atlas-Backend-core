## Siguiente Paso: Instalar PostgreSQL

Tienes 3 opciones:

### Opcion 1: Instalar con el script automatico
```powershell
.\install-postgresql.ps1
```

### Opcion 2: Descargar manualmente
Ve a: https://www.postgresql.org/download/windows/
Ejecuta el instalador y anota la contraseña que elijas

### Opcion 3: Usar PostgreSQL en la nube (mas rapido para probar)
Crea una cuenta gratuita en:
- https://www.elephantsql.com/ (PostgreSQL gratuito)
- https://supabase.com/ (incluye PostgreSQL)

Luego actualiza el archivo .env con las credenciales que te den.

## Despues de instalar PostgreSQL

1. Crear la base de datos:
```powershell
psql -U postgres -c "CREATE DATABASE atlas_db;"
```

2. Si usaste una contraseña diferente a "your_password_here", actualiza .env:
```
DB_PASSWORD=tu_contraseña_aqui
```

3. Ejecutar migraciones:
```powershell
npm run migrate
```

4. Iniciar el servidor:
```powershell
npm run dev
```

5. Probar la API:
Abre en el navegador: http://localhost:3000/api/v1/health
